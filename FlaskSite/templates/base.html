Project Explanation

This website, or web project, is constructed using the Python Flask framework; a lightweight and fast tool to use to create a website.

The end goal of this web project is to basically evolve it into a mockup online store (kind of like Amazon, although far smaller in scale), which will host tools like searching for products.

Once the mock site is completed, a complete rundown will be provided on the main portfolio site.
